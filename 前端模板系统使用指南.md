# 前端模板系统使用指南

## 概述

MusicOverlay 现在支持完全自定义的前端模板系统，您可以创建多个前端模板并在GUI中切换，实现不同的视觉效果。

## 目录结构

```
web/
├── templates/
│   ├── default/          # 默认模板
│   │   ├── index.html    # 完整 overlay
│   │   ├── cover.html    # 仅封面
│   │   ├── title.html    # 仅标题
│   │   ├── artist.html   # 仅艺术家
│   │   ├── album.html    # 仅专辑
│   │   ├── overlay.js    # WebSocket 客户端逻辑
│   │   └── custom/
│   │       └── user.css  # 用户自定义 CSS
│   └── custom1/          # 你的自定义模板
│       ├── index.html
│       └── ...
└── themes/               # 共享 CSS 主题（可选）
    └── ...
```

## 创建自定义前端模板

### 步骤1：创建新模板目录

在 `web/templates/` 目录下创建新文件夹，例如 `custom1`：

```
web/templates/custom1/
```

### 步骤2：复制默认模板作为起点（可选）

可以从 `default/` 复制所有文件到你的新模板作为起点：

```bash
cp -r web/templates/default/* web/templates/custom1/
```

### 步骤3：自定义你的模板

编辑 `custom1/` 中的文件以实现你的设计。

## 统一接口规范

所有前端模板都使用相同的后端接口，确保兼容性：

### WebSocket 接口

- **端点**: `ws://localhost:49090/ws`
- **推送的 JSON 格式**:
```json
{
  "title": "歌曲名",
  "artist": "艺术家",
  "album": "专辑",
  "cover": "data:image/jpeg;base64,...",
  "isPlaying": true,
  "sourceId": "smtc_system",
  "theme": { /* 主题配置对象 */ }
}
```

### HTTP API 接口

- `GET /api/now` - 获取当前播放信息（JSON）
- `GET /api/theme` - 获取当前主题配置（JSON）

### 页面路由

- `GET /` - 完整 overlay (`index.html`)
- `GET /cover` - 仅封面 (`cover.html`)
- `GET /title` - 仅标题 (`title.html`)
- `GET /artist` - 仅艺术家 (`artist.html`)
- `GET /album` - 仅专辑 (`album.html`)

### 静态文件路由

- `/overlay.js` - 从模板目录加载
- `/custom/*` - 从模板目录加载
- `/themes/*` - 从 `web/themes/` 共享目录加载

## 推荐的前端开发方法

### 方法1：使用 overlay.js

复用默认模板的 `overlay.js`，它提供了便捷的 WebSocket 封装：

```html
<script src="/overlay.js"></script>
<script>
  initOverlay({
    onMedia(data) {
      // 处理媒体信息更新
      console.log(data.title, data.artist, data.album, data.cover);
      applyTheme(data.theme);  // 应用主题配置
      setPlayState(data.isPlaying);
    }
  });
</script>
```

### 方法2：直接使用 WebSocket

完全自定义 WebSocket 连接：

```javascript
const ws = new WebSocket('ws://localhost:49090/ws');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  document.getElementById('title').textContent = data.title;
  document.getElementById('artist').textContent = data.artist;
  document.getElementById('album').textContent = data.album;
  document.getElementById('cover').src = data.cover;
};

ws.onerror = (error) => {
  console.error('WebSocket error:', error);
};
```

### 方法3：使用 HTTP 轮询（不推荐）

如果需要兼容不支持 WebSocket 的环境：

```javascript
async function updateMedia() {
  const response = await fetch('http://localhost:49090/api/now');
  const data = await response.json();
  // 更新界面
}

setInterval(updateMedia, 1000);  // 每秒轮询
```

## 在 GUI 中切换模板

1. 打开 MusicOverlay 设置窗口
2. 进入"状态"标签页
3. 在"前端模板"下拉框中选择你的模板
4. **重要**: 刷新 OBS 浏览器源以应用更改

## 示例：创建简单的文本 Overlay

创建 `web/templates/simple/index.html`:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <style>
    body {
      background: transparent;
      color: white;
      font-family: Arial;
      padding: 20px;
    }
    #title { font-size: 32px; font-weight: bold; }
    #artist { font-size: 24px; opacity: 0.8; }
    #album { font-size: 18px; opacity: 0.7; }
  </style>
</head>
<body>
  <div id="title">等待音乐...</div>
  <div id="artist"></div>
  <div id="album"></div>

  <script src="/overlay.js"></script>
  <script>
    initOverlay({
      onMedia(data) {
        document.getElementById('title').textContent = data.title || '等待音乐...';
        document.getElementById('artist').textContent = data.artist || '';
        document.getElementById('album').textContent = data.album || '';
      }
    });
  </script>
</body>
</html>
```

## 高级技巧

### 1. 使用主题配置

后端推送的 `theme` 对象包含用户在 GUI 中配置的样式信息，可以动态应用：

```javascript
onMedia(data) {
  if (data.theme) {
    document.documentElement.style.setProperty('--title-color', data.theme.title.color);
    document.documentElement.style.setProperty('--title-size', data.theme.title.size + 'px');
    document.documentElement.style.setProperty('--album-color', data.theme.album.color);
    document.documentElement.style.setProperty('--album-size', data.theme.album.size + 'px');
  }
}
```

### 2. 响应播放状态

```javascript
function setPlayState(isPlaying) {
  if (isPlaying) {
    coverImg.style.animationPlayState = 'running';
  } else {
    coverImg.style.animationPlayState = 'paused';
  }
}
```

### 3. 处理封面动画

```css
@keyframes spin {
  to { transform: rotate(360deg); }
}

#cover {
  animation: spin 8s linear infinite;
  animation-play-state: paused;
}
```

## 故障排除

### 模板没有在下拉框中显示

- 确保模板目录位于 `web/templates/` 下
- 重启 MusicOverlay 程序

### 切换模板后没有效果

- 在 OBS 中刷新浏览器源（右键 → 刷新）
- 检查浏览器控制台是否有错误

### WebSocket 连接失败

- 确认 MusicOverlay 正在运行
- 检查端口是否被占用（默认 49090）
- 检查防火墙设置

## 总结

前端模板系统提供了完全的灵活性，你可以：

- 创建任意数量的前端模板
- 使用任何前端技术栈（Vue, React, 纯 HTML/CSS/JS 等）
- 保持与后端的统一接口
- 在 GUI 中轻松切换

祝你创作出精彩的音乐 Overlay！
